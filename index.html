Description du projet

Par Louise, Albain, David, le 26/02/2020 à 17:00


Le jeu se base sur la forme standard d’un jeu d’arcade, dont le but est d’avancer avec son personnage pour atteindre un objectif particulier, tel Mario Bros. Pour notre projet, nous avons décidé d’élaborer un jeu basé sur un-e étudiant-e de la HES-SO, à Sierre, qui étant en retard pour les cours, doit courir dans les couloirs pour atteindre sa classe. Sur le trajet, il-elle va rencontrer divers obstacles qui vont le-la ralentir.

Au lancement du jeu, l’utilisateur-trice a le choix du personnage : Albain, David ou Louise, les trois concepteurs du projet. Il-elle a ensuite le choix entre différents niveaux de difficulté. Avant que la course commence, le temps à disposition ainsi que la classe à atteindre, sont affichés. Le jeu commence directement à l’étage numéro 1.

L’utilisateur-trice joue avec les flèches de son clavier pour avancer, ainsi qu’avec la barre d’espace pour sauter.

Le code source de l'application est disponible sur le github du projet.
